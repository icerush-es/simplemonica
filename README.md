# MoniCard 非公式クライアント（Android）

> ⚠️ **非公式アプリだよ。** MoniCard（魔力卡）のメーカー・販売元とは一切関係ないよ。
> 本アプリはBLEで MoniCardと通信する有志製のクライアントだよ。利用は自己責任でよろしくね。

MoniCardに画像・動画を送ったり、名刺・興味タグ・各種設定を Android から操作できるアプリだよ。

## 主な機能
- 画像を送る（トリミング・拡大縮小・回転）
- 動画/GIFを送る（開始位置・長さ・fps・容量見積り）
- 名刺（自己紹介文）の設定・受信名刺の確認
- 興味タグ／合言葉の設定
- 画像切換間隔・各種トグルの設定
- デバイス情報の表示、デバイス名の変更
- 複数台の MoniCard を選んで接続

## 必要環境
- Android 8.0（API 26）以上
- Bluetooth（BLE）対応端末
- MoniCard本体

## インストール方法
1. このページの右側 **Releases** から最新の `.apk` をダウンロード
2. スマホで「提供元不明のアプリ / 不明なアプリのインストール」を許可
3. ダウンロードした APK を開いてインストール
4. 初回起動時に **Bluetooth / 付近のデバイス** の権限を許可

## 権限について
- **付近のデバイス（Bluetooth）**：MoniCardを探して接続するために使用するよ。位置情報の取得には使わないよ。
